    JMap.Data.Photo.getSelectedPhotoId

    Return the current selected/displayed photo ids.

    Return undefined if no photo is selected.

    This function is the equivalent of :

    JMap.Data.Photo
   .getPhotoDescriptors()
   .map(photoDescriptor => photoDescriptor.id)
