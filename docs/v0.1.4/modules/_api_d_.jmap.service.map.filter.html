    JMap.Service.Map.Filter.applyHasNotAttribute

    For a layer, show on map only the features that hasn't the attribute defined in their properties. If the property is defined but value is undefined, the feature will not be shown.

    example
    JMap.Service.Map.Interactor.applyHasNotAttribute(4, "name")
